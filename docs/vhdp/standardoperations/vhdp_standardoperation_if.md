## Definition

Executes an operation if a condition is met.

## Examples

```vhdp
If (a = 0) 
{ 
	a := 1; 
} 
Elsif (a = 1) 
{ 
	a := 2; 
} 
Else 
{ 
	a := 0; 
}
```
