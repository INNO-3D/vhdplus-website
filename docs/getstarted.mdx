### Drivers

You need to install the drivers according to your programmer at hand in order to program your FPGA. Our VHDPlus Core features an ARROW USB Programmer because of its additional USB to UART interface.

Follow **[this guide](/docs/getstarted/drivers)** to download and install the driver for the **Arrow USB Programmer**.

### Connect VHDPlus IDE with Quartus

For direct compiling and programming of VHDPlus IDE a connection via Quartus is necessary. 
If Quartus is installed in default directory no further steps are needed. 
 Otherwise an adjustment of the Quartus' path can be realized by opening the VHDPlus IDE and navigating through `Extras -> Options -> General`

![Select Quartus Path](/img/getstarted/QuartusPath.png)

When Quartus is detected successfully the boundary turns green.

#### Using the new Package Manager (recommended)
1. Open the Package Manager `Extras` -> `Package Manager`
2. Install `GHDL` and `GTKWave` and wait for them to finish

#### Manual installation
1. Download and Install <a href="http://ghdl.free.fr/ghdl-installer-0.29.1.exe" target="_blank">GHDL</a>
2. Download an extract <a href="https://sourceforge.net/projects/gtkwave/files/gtkwave-3.3.100-bin-win64/gtkwave-3.3.100-bin-win64.zip/download" target="_blank">GTK Wave</a>
3. Specify the path for GHDL and GTKWave executables in `Extras` -> `Settings` -> `Simulator`


  </TabItem>

2. Download an install GTKWave with `sudo apt-get install gtkwave` or follow <a href="http://gtkwave.sourceforge.net/" target="_blank">these</a> instructions

### Setup Simulation with ModelSim (Optional)

1. Download and install <a href="https://download.altera.com/akdlm/software/acdsinst/18.1std/625/ib_installers/ModelSimSetup-18.1.0.625-windows.exe" target="_blank">ModelSim for Windows</a> or <a href="http://download.altera.com/akdlm/software/acdsinst/18.1std/625/ib_installers/ModelSimSetup-18.1.0.625-linux.run" target="_blank">ModelSim for Linux</a>
2. Specify the path for the `modelsim_ase` folder in `Extras` -> `Settings` -> `Simulator`
3. Run the simulation in the GHDP viewer or with a right click on a VHDL file

Learn more about simulation with ModelSim [here](/docs/getstarted/modelsim)

## Need Help?

Starting to program FPGAs can be quite tricky.
Here are a few options that can help you get started.

- Join our [Discord](https://discord.gg/NCN9VAh)! We are happy to help with any problems and to answer questions.
- Ask questions on [Stack Overflow](https://stackoverflow.com/questions/tagged/vhdp)
- Check out our [YouTube Channel](https://www.youtube.com/channel/UC7qiOvlaBSiWyAb7R1xTaEw). We upload useful tutorials frequently.
- Check out [Community Projects](/docs/community/overview#example-projects-with-tutorial). See what is possible with VHDPlus and learn how to create those Projects yourself.
